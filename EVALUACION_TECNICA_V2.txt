───────────────────────────────────────────────────────────────────────────────
1. MÓDULO DE ANÁLISIS DE DOCUMENTOS
───────────────────────────────────────────────────────────────────────────────

Crear una pantalla web para la carga de documentos en formato PDF, JPG o PNG
con las siguientes funcionalidades:

┌─────────────────────────────────────────────────────────────────────────────┐
│ A. CLASIFICACIÓN AUTOMÁTICA                                                │
└─────────────────────────────────────────────────────────────────────────────┘

Clasifica automáticamente los documentos cargados en:

    • FACTURA (Invoice)
      └─ Contiene datos económicos/financieros

    • INFORMACIÓN (Information)
      └─ Texto general


┌─────────────────────────────────────────────────────────────────────────────┐
│ B. EXTRACCIÓN AUTOMÁTICA DE DATOS                                          │
└─────────────────────────────────────────────────────────────────────────────┘

Si es FACTURA, extraer:
    • Cliente (Client)
      └─ Nombre y dirección

    • Proveedor (Provider)
      └─ Nombre y dirección

    • Número de factura (Invoice number)
    • Fecha (Date)
    • Productos (Products)
      └─ Cantidad (Quantity)
      └─ Nombre (Name)
      └─ Precio unitario (Unit price)
      └─ Total (Total)

    • Total de la factura (Invoice total)


Si es INFORMACIÓN, extraer:
    • Descripción (Description)
    • Resumen del contenido (Content summary)
    • Análisis de sentimiento (Sentiment analysis)
      └─ Positivo (Positive)
      └─ Negativo (Negative)
      └─ Neutral (Neutral)


┌─────────────────────────────────────────────────────────────────────────────┐
│ C. RECOMENDACIÓN DE IA                                                     │
└─────────────────────────────────────────────────────────────────────────────┘

Sugerencia: Utilizar herramientas como:
    • AWS Textract
    • Azure Cognitive Services
    • OpenAI

Para la extracción de datos.


───────────────────────────────────────────────────────────────────────────────
2. MÓDULO HISTÓRICO
───────────────────────────────────────────────────────────────────────────────

Crear una pantalla para mostrar un log de eventos registrados:

┌─────────────────────────────────────────────────────────────────────────────┐
│ REGISTRA                                                                    │
└─────────────────────────────────────────────────────────────────────────────┘

    • ID del evento (Event ID)
    • Tipo (Type)
      └─ Carga de documento (Document upload)
      └─ IA (AI)
      └─ Interacción de usuario (User interaction)
    • Descripción del evento (Event description)
    • Fecha y hora (Date and time)


┌─────────────────────────────────────────────────────────────────────────────┐
│ FUNCIONALIDADES                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

    • Filtros por tipo, descripción o rango de fechas
      (Filters by type, description, or date range)

    • Exportación del contenido a Excel
      (Export content to Excel)

                              RESUMEN
